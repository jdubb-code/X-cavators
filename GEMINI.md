# Gemini Code Assistant Context

This document provides context for the Gemini Code Assistant to understand and effectively assist with the **X-cavators** project.

## Project Overview

**X-cavators** is a simple SwiftUI application for macOS. The project is in its initial stage, displaying only a logo on a black background. The main technologies used are Swift and SwiftUI.

The project is structured as a standard Xcode project:

*   `X-cavators/`: Contains the main application source code.
    *   `X_cavatorsApp.swift`: The entry point of the application.
    *   `ContentView.swift`: The main view of the application.
    *   `Assets.xcassets`: Contains the application's assets, such as the logo.
*   `X-cavators.xcodeproj/`: The Xcode project file.

## Building and Running

To build and run the project, open `X-cavators.xcodeproj` in Xcode and click the "Run" button.

## Development Conventions

*   The project follows the standard Swift and SwiftUI conventions.
*   The code is written in a clear and concise style.
*   The project uses the standard Xcode project structure.
